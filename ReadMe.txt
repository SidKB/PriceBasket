=============PRICE BASKER =============

This is a simple java command line based application written in Java8.
As order to implement the features decorator pattern is been used.
Catalogue items and their corresponding values are defined in the message.properties file.more items can be added there
This design also supports futures enhancement.

Maven could be used Maven to generate deployable artifacts.
Junits/Mochito for test cases.
Spring DI in order object injection ( code is right now tightly coupled as no such tech been used)


Run the Application:
===================
Post comilation Run :: javac com/bjss/PriceBasket "pass the items spaces sepparated"
